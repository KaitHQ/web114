Questions

Question 1: When would you use a switch statement in your projects?

Using switch statements are very useful when you have multiple possible answers and you want to tidy up your code instead of having several if, else if, else if, else if, etc etc. Using a switch statement eliminates the need for the if, else if, else statements, the curly brackets, and it overall makes the code look tidier.
